The Leaf Q mark.

A mint leaf whose central vein curls into a question mark. One idea, two readings at once — the leaf signals mint (freshness, learning in bite-size), the stem hook signals quiz (the question being asked). This document is a complete handoff: final SVG, React component, color tokens, sizes, usage rules, and the animated splash screen (§13). v1.2 update: the vein and dot are now slate (#334155) instead of white/mint-deep so the mark reads on any background — paper, app-light, or dark sidebar.

09 · React component

Drop this at src/components/QuizMintLogo.tsx. Props for size and variant cover all four authorized forms.

src/components/QuizMintLogo.tsx 
// QuizMintLogo — the Leaf Q mark.
// The leaf is a mint leaf; its central vein is a question-mark stem.
type Variant = 'default' | 'mono-ink' | 'mono-white';

interface Props {
  size?: number;         // pixel size (square). default 32.
  variant?: Variant;     // default | mono-ink | mono-white
  className?: string;
  title?: string;        // a11y label. omit for decorative use.
}

export function QuizMintLogo({
  size = 32,
  variant = 'default',
  className,
  title = 'QuizMint',
}: Props) {
  const leafA = variant === 'default' ? '#10B981'
              : variant === 'mono-white' ? '#FFFFFF'
              : '#0A0A0C';
  const leafB = variant === 'default' ? '#059669' : leafA;
  const vein  = variant === 'mono-ink' ? '#FFFFFF'
              : variant === 'mono-white' ? '#0A0A0C'
              : '#334155';   // slate — reads on any bg
  const dot   = variant === 'default' ? '#334155' : leafA;
  const veinOpacity = variant === 'default' ? 0.85 : 1;

  return (
    <svg
      xmlns="http://www.w3.org/2000/svg"
      viewBox="0 0 100 100"
      width={size}
      height={size}
      role="img"
      aria-label={title}
      className={className}
    >
      <title>{title}</title>
      <path d="M50 10 C 74 10, 88 28, 88 48 C 88 63, 76 74, 60 74 L 50 74 Z" fill={leafA}/>
      <path d="M50 10 C 26 10, 12 28, 12 48 C 12 63, 24 74, 40 74 L 50 74 Z" fill={leafB}/>
      <path
        d="M50 14 L 50 64 C 50 74, 60 76, 60 84"
        fill="none"
        stroke={vein}
        strokeWidth={5.5}
        strokeLinecap="round"
        opacity={veinOpacity}
      />
      <circle cx={60} cy={92} r={3.4} fill={dot}/>
    </svg>
  );
}

13 · Animated splash screen · "Draw On"

Shown on first load and on reload, while the app boots. The leaf draws on stroke-first (right half, then left, overlapping slightly), the filled color flushes in, the vein writes like a pen, the question-dot pops in, and finally the wordmark fades up. Splash component

Drop-in React component. Renders a full-viewport overlay, plays the animation, then fades itself out and unmounts. Accepts a minDurationMs to guarantee the user sees the full animation even on a fast boot, and an optional onDone callback.

src/components/QuizMintSplash.tsx 
import { useEffect, useState } from 'react';
import './QuizMintSplash.css';

interface Props {
  minDurationMs?: number;   // keep splash visible at least this long. default 2800.
  onDone?: () => void;      // called after fade-out completes
}

export function QuizMintSplash({ minDurationMs = 2800, onDone }: Props) {
  const [phase, setPhase] = useState<'playing' | 'fading' | 'done'>('playing');

  useEffect(() => {
    const t1 = setTimeout(() => setPhase('fading'), minDurationMs);
    const t2 = setTimeout(() => { setPhase('done'); onDone?.(); }, minDurationMs + 400);
    return () => { clearTimeout(t1); clearTimeout(t2); };
  }, [minDurationMs, onDone]);

  if (phase === 'done') return null;

  return (
    <div className={`qm-splash play ${phase === 'fading' ? 'fade-out' : ''}`} aria-hidden="true">
      <div className="qm-ambient" />
      <div className="qm-logo">
        <svg viewBox="0 0 100 100" width={140} height={140} style={{ overflow: 'visible' }} aria-label="QuizMint">
          <path className="qm-leaf-r" d="M50 10 C 74 10, 88 28, 88 48 C 88 63, 76 74, 60 74 L 50 74 Z" fill="#10B981"/>
          <path className="qm-leaf-l" d="M50 10 C 26 10, 12 28, 12 48 C 12 63, 24 74, 40 74 L 50 74 Z" fill="#059669"/>
          <path className="qm-vein" d="M50 14 L 50 64 C 50 74, 60 76, 60 84"
                fill="none" stroke="#334155" strokeWidth={5.5} strokeLinecap="round" opacity={0.85}/>
          <circle className="qm-dot" cx={60} cy={92} r={3.4} fill="#334155"/>
        </svg>
        <div className="qm-word">Quiz<span style={{ color: '#10B981' }}>Mint</span></div>
        <div className="qm-loader"><div className="qm-loader-bar" /></div>
      </div>
    </div>
  );
}

Mounting the splash

Wrap the app root in main.tsx so the splash is the first thing the user sees, even before React hydrates the rest of the tree. Use sessionStorage so repeat navigations within the same session skip it — the user sees it on reload / first open, not on every client-side nav.

src/main.tsx · wrap App 
import { useState } from 'react';
import { QuizMintSplash } from './components/QuizMintSplash';
import App from './App';

function Root() {
  // Show splash on first load of this session only.
  const [showSplash, setShowSplash] = useState(() => {
    if (typeof window === 'undefined') return false;
    const seen = sessionStorage.getItem('qm-splash-seen');
    return !seen;
  });

  return (
    <>
      <App />
      {showSplash && (
        <QuizMintSplash onDone={() => {
          sessionStorage.setItem('qm-splash-seen', '1');
          setShowSplash(false);
        }} />
      )}
    </>
  );
}

ReactDOM.createRoot(document.getElementById('root')!).render(<Root />);
